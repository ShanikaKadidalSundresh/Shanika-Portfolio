## Run Locally

1. Clone the repository:
   git clone <your-repo-link>

2. Go into the project folder:
   cd shanika-portfolio

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

The app will run at:
http://localhost:5173
